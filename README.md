# 凯茜的模组无尽模式支持

打完第三关，发现强力卡差了一点点没养成？意犹未尽？

感觉全挑战也太简单？

本Mod可以解决前述两个问题
1. 将凯茜的模组改为10关 （9关普通关+1关莱茜关）
2. 后边的Boss关的开启挑战14时的灰熊大队做了增强，第N场Boss战里的灰熊大队的攻击血量\*N，例如：
第8场Boss战里的灰熊大队的攻击、血量\*8，变成了32攻48血。

因此，本Mod开启灰熊挑战，难度将再上升一个等级。
